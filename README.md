# SAPIR-Net Phase 2: Antibiotics

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.20376649.svg)](https://doi.org/10.5281/zenodo.20376649)
[![License: CC BY 4.0](https://img.shields.io/badge/License-CC%20BY%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by/4.0/)

A reproducible stochastic network simulation of upstream concentration and cascading-disruption vulnerability in the U.S. generic antibiotic supply chain.

---

## Overview

SAPIR-Net Phase 2 extends the three-layer directed-graph network model from [Phase 1 (oncology)](https://doi.org/10.5281/zenodo.19549343) to four essential U.S. generic antibiotics: amoxicillin, amoxicillin/clavulanate, azithromycin, and doxycycline. Calibrated to UN Comtrade bilateral trade data (HS 294110, 294130, 294150, 294190; calendar years 2018–2025) and to a convergent source stack (National Academies 2022; Schondelmeyer U.S.-China Economic and Security Review Commission testimony 2025; U.S. Pharmacopeia 2024 key-starting-material analysis; Socal et al. 2025 in *JAMA Health Forum*), Monte Carlo simulation (N = 10,000 iterations per scenario, fixed pseudo-random generator seed = 42) identifies:

- A **100% probability of severe shortage** for every drug in the scope under a cascading upstream shock scenario, structurally guaranteed at the source-stack-calibrated chemical-class lower band endpoints.
- A **dual-chokepoint signature** in amoxicillin/clavulanate: 65.4% probability of severe shortage under a global logistics scenario, against 20.9% for single-input drugs — identifying combination antibiotic products as a structurally compounded supply chain risk category.

The associated manuscript ("Network Modeling of U.S. Generic Antibiotic Supply Chain Concentration and Disruption") is targeted to *JAMA Health Forum* as an Original Investigation / Decision Analytical Model. Phase 2 is the antibiotics component of the integrated SAPIR-Net research program, which also includes the [National Pharmaceutical Asset Optimization and Resilience Framework (NPAORF)](https://doi.org/10.5281/zenodo.19310969) and the forthcoming National Pharmaceutical Intelligence Network.

## Reproduction

### Prerequisites

- Python 3.12 (developed and tested against 3.12.3; minor versions ≥ 3.12 should work; Python 3.14 is untested but expected to work — if you encounter package compatibility issues with the pinned versions on 3.14, use a Python 3.12 virtual environment)
- ~50 MB free disk space
- ~5 minutes wall-clock time for the full pipeline

### Setup

```bash
# Clone the repository
git clone https://github.com/fahimkazmi911/sapir-net-phase2-antibiotics.git
cd sapir-net-phase2-antibiotics

# Create and activate a virtual environment (recommended)
python3 -m venv venv
source venv/bin/activate   # on Windows: venv\Scripts\activate

# Install pinned dependencies
pip install -r requirements.txt
```

### Run the pipeline

The modules are designed to be run in order. Each consumes the prior module's output and writes its own outputs to `outputs/`.

```bash
# Module 1: graph construction and HHI concentration analysis
python modules/sapir_phase2_module1_graph_hhi.py

# Module 2: disruption engine (scenario classes, sampler validation)
python modules/sapir_phase2_module2_disruption_engine.py

# Module 3: Monte Carlo simulation (primary + sensitivity panel)
python modules/sapir_phase2_module3_monte_carlo.py

# Module 3 partial-correlation extension (S4 Gaussian-copula sensitivity)
python modules/sapir_phase2_module3_correlation_run.py

# Module 4A: render Figures 1, 2, 3
python modules/sapir_phase2_module4a_visualizations.py

# Figure 4: render Figure 4 (partial-correlation density)
python modules/sapir_phase2_figure4_correlation.py
```

### Verify byte-exact reproduction

Under the pinned `requirements.txt` environment, RNG seed 42 reproduces all numerical outputs to byte-exact tolerance. To verify:

```bash
# Linux / macOS
md5sum outputs/*.csv

# Compare against these reference hashes:
# 8b416a6fd22c3120887aad9c0ce6f63b  outputs/phase2_hhi_analysis.csv
# 1456e102c6d48aaaaa9c8feeebd962d0  outputs/phase2_hhi_aggregate_2020_2024.csv
# 0f5fcdcb230deb84d744723b389a4a7b  outputs/phase2_calibration_table.csv
# eb14ae9444d649e996603d625f6e5dbc  outputs/phase2_monte_carlo_primary.csv
# 2b7184ee7766806bd1afd6b7cc403495  outputs/phase2_monte_carlo_sensitivity.csv
# fb30103771e95ed44fd5b89282f1bcf5  outputs/phase2_monte_carlo_correlation.csv
```

If any hash differs, you are likely running with a different package version than the pinned set. Check `pip show <package>` against the versions in `requirements.txt`.

## Methodology summary

The SAPIR-Net model is a three-layer directed graph implemented in Python using NetworkX:

- **Layer 1 (geopolitical sources)**: China, India, Rest of World
- **Layer 2 (chemical commodity APIs)**: four nodes corresponding to HS 294110 (penicillin), 294130 (tetracycline), 294150 (erythromycin/macrolide), 294190 (other antibiotics including clavulanic acid)
- **Layer 3 (essential medicines)**: four nodes corresponding to amoxicillin, amoxicillin/clavulanate, azithromycin, doxycycline

Edges from Layer 1 to Layer 2 carry 2020–2024 USD-aggregate bilateral trade weights from UN Comtrade. Edges from Layer 2 to Layer 3 carry binary obligate-precursor flags, with one combination-drug-specific propagation rule: amoxicillin/clavulanate's capacity is the *multiplicative joint* of the two independent fermentation chokepoints it depends on (6-aminopenicillanic acid via penicillin G fermentation; clavulanic acid via *Streptomyces clavuligerus* fermentation). This is the single Phase 2–specific methodological extension over Phase 1.

Three disruption scenarios are simulated:

- **Scenario A** (Direct Export Ban) — deterministic: China and India direct exports zeroed across all four HS codes.
- **Scenario B** (Cascading Upstream Shock) — China direct exports zeroed, Rest-of-World per-HS supply degraded by a Pareto-rescaled fraction within chemical-class `row_china_exposure` bands.
- **Scenario C** (Global Logistics Chokepoint) — uniform log-normal capacity reduction applied to all Layer 1 → Layer 2 edges simultaneously.

Four sensitivity analyses are reported: S1 / S2 (parameter-band endpoints), S3 (sub-recent baseline window for HS 294150), and S4 (Gaussian-copula partial-correlation on the multiplicative-joint independence assumption).

Full methodology, calibration source stack, distribution validation, complete sensitivity panel, and partial-correlation Gaussian-copula derivation are documented in the manuscript supplementary materials (`paper/SAPIR-Net_Phase2_Antibiotics_SupplementaryMaterials_v1_3.md`).
